# React Fundamentos - Products Dashboard

Your site is live at https://lauramarianagt-alt.github.io/pr-ctica-fundamentos-React/

Aplicación desarrollada con React + TypeScript como práctica final del módulo de Fundamentos de React.

## Características

- Autenticación mediante token JWT.
- Rutas protegidas.
- Listado de productos.
- Búsqueda por nombre.
- Filtro de productos en oferta.
- Visualización de detalle de producto.
- Creación de nuevos productos.
- Edición de productos existentes.
- Eliminación de productos.
- Logout de usuario.

## Tecnologías

- React
- TypeScript
- Vite
- React Router
- CSS
- JSON Server

# Funcionalidades implementadas

### Autenticación

- Login de usuario.
- Almacenamiento de token.
- Protección de rutas privadas.
- Logout.

### Productos

- Listado de productos.
- Filtro por nombre.
- Filtro de ofertas.
- Creación de productos en `/products/new`.
- Edición mediante formulario modal.
- Eliminación de productos.
- Vista de detalle.
